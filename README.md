# Auditoria de Pedágios

Projeto em Python para automatizar a conferência de faturas de pedágio com relatórios de auditoria, identificando divergências e exportando resultados para Excel.

## Tecnologias
- Python (pandas, openpyxl)
- Excel
- Automação de processos financeiros

## Funcionalidades
- Leitura de relatórios de auditoria e faturas
- Comparação automática de valores
- Exportação de divergências para Excel
- Estrutura modular para reutilização mensal

## Objetivo
Garantir auditoria de passagens de pedágio de forma automatizada, confiável e escalável.
